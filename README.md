# Greenhouse Monitoring System

A comprehensive smart agriculture monitoring system with AI-powered irrigation control.

## Features

- **Real-time Monitoring**: Temperature, humidity, soil moisture (3 zones)
- **AI-Powered Irrigation**: Q-Learning + Linear Regression hybrid system
- **Resource Tracking**: Water and energy consumption monitoring
- **Dual Interface**: Kiosk mode for users + Admin dashboard
- **Data Persistence**: SQLite database with advanced analytics
- **Remote Control**: Frontend API synchronization
- **Automated Scheduling**: ML-based irrigation decisions
